Web Design Definitions

File
A logical collection of information stored under a simple name
Information
Words, numbers, pictures, and sounds which have meaning to us
Directory
A logical collection of files stored under a single name
Can also be referred to as a folder
JPEG
Joint Photographic Experts Group
A format for compressing image files
GIF
Graphics Interchange Format
A lossless format for image files that suppports both animated and static images
PNG
Portable Network Graphics
A lossless format for simple images for flat areas of colour
PDF
Portable Document Format
A file format used to preserve the original document regardless of what is used to view it
