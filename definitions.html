Definitions

File
A logical collection of information stored under a simple name
Information
Words, numbers, pictures, and sounds which have meaning to us
Directory
A logical collection of files stored under a single name
Can also be referred to as a folder
JPEG
Joint Photographic Experts Group
A format used to compress image files
GIF
Graphics Interchange Format
A lossless format for both animated and static images
PNG
Portable Network Graphic
A lossless format used to save simple images with flat areas of colour
PDF
Portable Document Format
Used to preserve the format of a document regardless of the device or software used to view it
